Academic projects

Here are some of the most important projects I worked on:

  • I - Machine Learning & AI :

    1 - Multi-target tracker :

    October 2016 - January 2017

    Consider a random number of targets that appear at random positions and move according to a known transition model. Given a radar that generates observed positions (with noise) and a random number of false alarms, we would like to track the true positions. This problem cannot be solved with a classic particle filter since the dimension of the position vectors changes in time. Introduced in (Vo et al, 2005), the PHD filter is built on FISST (Finite Set Statistics) theory and designed to solve these problems.
    - Results paper -


    2 - Random features for Kernel approximation :

    April 2017 - May 2017

    Review and implementation of kernel approximation using random features. Particularly the work of (J. Pennington et al, 2015) on polynomial kernels and that of (Rahimi and Recht, 2007). Review and Python code can be found in my Git repository.


  • II - Genomics :

    Data analysis framework :

    Spring - Summer 2016 - INRIA

    Study of resistance to chemotherapy in Glioblastoma using RNA-seq and single cell data. I created a python GUI with PyQt for data analysis. See my section software. The internship final paper is available here.


  • III - Information theory :

    LDPC Codes :

    Spring 2016

    Implementation of LDPC codes construction, coding and decoding (using the sum-product algorithm). A beautiful mixture of linear algebra and probability theory. The released Python package pyLDPC and several tutorials are available at PyPI.
