Academic projects

Here are some of the most important projects I worked on:

  • Machine Learning & AI

    October 2016 - January 2017

    Multi-target tracker - Consider a random number of targets that appear at random positions and move according to a known transition model. Given a radar that generates observed positions (with noise) and a random number of false alarms, we would like to track the true positions. This problem cannot be solved with a classic particle filter since the dimension of the position vectors changes in time. Introduced in (Vo et al, 2005), the PHD filter is built on FISST (Finite Set Statistics) theory and designed to solve these problems. Results paper .

    January 2017 - present

    NLP - Sentiment analysis - Confrontation of bag of words and word2vec model (Micholov, 2014) in predicting the satisfaction of a reader given his/her review of a book.

  • Genomics

    Spring - Summer 2016 - INRIA

    Data analysis framework - Study of resistance to chemotherapy in Glioblastoma using RNA-seq and single cell data. I created a python GUI with PyQt for data analysis. See my section software. The internship final paper is available here.

  • Information theory

    Spring 2016

    LDPC Codes - Implementation of LDPC codes construction, coding and decoding (using the sum-product algorithm). A beautiful mixture of linear algebra and probability theory. The released Python package pyLDPC and several tutorials are available at PyPI.
