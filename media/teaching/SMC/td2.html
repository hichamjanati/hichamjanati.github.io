1 - Laplace Generator with inverse sampling¶

Soit $U \sim \mathcal U([0, 1])$ et $X$ une var suivant une loi $\mu$, de fonction de rpartition $F$ admettant une rciproque $F^{-1}$. Le <a href=https://en.wikipedia.org/wiki/Inverse_transform_sampling> thorme d'inversion </a> assure que la var $F^{-1}(U)$ suit la loi $\mu$. Ainsi, pour gnrer un chantillon $X_1, \dots, X_n$ suivant la loi de Laplace, il suffit d'inverser sa fonction de rpartition et l'appliquer U.

Pour une loi de la place on a:

$$F(t) = \left\{ \begin{array}{l} \frac{1}{2} e^{x} \text{ si }x \leq 0 \\ 1 - \frac{1}{2}e^{-x} \text{ si } x \geq 0 \\ \end{array} \right. $$

Et donc:

$$\forall u \in [0, 1] \qquad F^{-1}(u) = \left\{ \begin{array}{l} \log(2u) \text{ si } u \leq \frac{1}{2} \\ \log(2 - 2u) \text{ si } u \geq \frac{1}{2} \\ \end{array} \right. $$

def laplace_list(n, seed=None):
    """Laplace generator.
    n: int.
        number of samples.
    seed: int (optional).
        random seed initialization.
    """
    # init generator
    rnd = np.random.RandomState(seed)
    L = []
    for i in range(n):
        u = rnd.rand()
        if u <= 0.5:
            L.append(np.log(2 * u))
        else:
            L.append(- np.log(2 - 2 * u))
    return L

def laplace(n, seed=None):
    """Laplace generator.
    n: int.
        number of samples.
    seed: int (optional).
        random seed initialization.
    """
    rnd = np.random.RandomState(seed)
    U = rnd.rand(n) # generate random sample U of size n
    mask = U < 0.5 # boolean array 
    L1 = mask * np.log(2 * U) # when u < 0.5 (not concerned entries are zero, the others are 0)
    L2 = - (1 - mask) * np.log(2 - 2 * U) # when u > 0.5
    L = L1 + L2
    return L

2 - Laplace Generator with reject algorithm¶

En se basant sur la loi de Laplace, on gnre un chantillon suivant la loi Normale centre rduite. On cherche donc majorer pour tout $x$ la densit gaussienne: $$ f(x) = \frac{1}{2\pi} \exp(-x^2 / 2) $$ par une constante M fois la densit de Laplace: $$ M g(x) = M \frac{1}{2} \exp{- |x|} $$.

Pour gnrer N observations:

  • List = []
  • for i in [1, n]: 
      1. x sampled from g
  2. u sampled from U[0, 1]
  3. if u < f(x) / Mg(x):
          add x to List
     else: 
         repeat 1., 2., 3.

Comme la probabilit d'acceptation est donne par 1 / M (<a href=http://www.columbia.edu/~ks20/4703-Sigman/4703-07-Notes-ARM.pdf> Proof here </a>), on cherche ce que M soit la plus petite valeure possible: On prend donc: $$ M = \sup_x f(x)/g(x) = \sqrt{\frac{2}{\pi}}\sup_x \exp(-x^2 / 2 + |x|) = \sqrt{\frac{2 e}{\pi}} $$

Crons deux fonctions f et g (les densits) en utilisant l'expression lambda:

def gaussian(n, f=f, g=g, M=M, seed=None):
    """Gaussian generator with reject algorithm.
    n: int.
        number of samples.
    f: callable.
        density we want to sample from.
    g: callable.
        density we use as surrogate.
    M: float > 1.
        bound constant f < M g.
    seed: int (optional).
        random seed initialization.
    """
    rnd = np.random.RandomState(seed)
    Y = np.zeros(n) # liste que l'on remplit avec des x ~ N(0, 1)
    j = 0 # compteur du nombre total d'essais
    for i in range(n):
        while(True):
            j += 1
            x = rnd.laplace()
            u = rnd.rand()
            if M * g(x) * u <= f(x):
                break
        Y[i] = x
    return Y, j
