$$ x_{k+1} = x_k - \alpha \nabla f(x_k) $$

$\alpha$ est le pas de descente, ou en anglais appelé le step-size ou le learning rate en machine learning.

def gradient_descent(alpha, n_iters=15):
    z = np.array([-4, 4])
    zs = [z]
    for ii in range(n_iters):
        z = z - alpha * gradient(z[0], z[1])
        zs.append(z)
    zs = np.array(zs)
    return zs
